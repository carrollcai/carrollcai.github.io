event loop原理

2018-09-27

浏览器event loop

  1. 所有的同步任务都在主线程内执行，形成一个执行栈（execute stack）。
  2. 所有的异步任务都在任务队列（task queue）里，一旦有了返回结果，就在任务队列里加入一个事件。
  3. 执行栈中的同步任务执行完毕后，将任务队列有结果的事件加入到执行栈中，执行同步任务时有异步任务，再加入到任务队列中。
  4. 主线程不断重复上面的第三步。

nodejs event loop

  1. V8引擎解析js代码。
  2. 解析后的代码，调用Node API。
  3. libuv执行Node API，它将不同的任务分配给不同的线程，形成一个event loop，最终异步的将返回结果返回给V8引擎。
  4. V8引擎将结果返回给用户。
